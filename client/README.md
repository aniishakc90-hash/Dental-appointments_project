# Dental Care Client

##How to run the project

1. Start the backend API(server)
    Open terminal and run:
    cd ../api
    npm start or node server.js

    The server will run at"
    http://localhost:3000

2. Open frontend:
    Open client/index.html using Live server

##Featurs
  -View appointments (Get request)
  -Add appointments
  -Edit appointment
  -Delete appointment

##Notes
  -Backend server must be running before using the client.
  -The client communicates with the backend using Fetch API.
  -Data is retrived from the API endpoint.
